344     # Allows access to the object attributes, which are held in the <tt>@attributes</tt> hash, as though they
 
345     # were first-class methods. So a Person class with a name attribute can use Person#name and
 
346     # Person#name= and never directly use the attributes hash -- except for multiple assigns with
 
347     # ActiveRecord#attributes=. A Milestone class can also ask Milestone#completed? to test that
 
348     # the completed attribute is not +nil+ or 0.
 
349     #
 
350     # It's also possible to instantiate related objects, so a Client class belonging to the clients
 
351     # table with a +master_id+ foreign key can instantiate master through Client#master.
 
352     def method_missing(method_id, *args, &block)
 
353       method_name = method_id.to_s
 
354       if match = match_attribute_method?(method_name)
 
355         guard_private_attribute_method!(method_name, args)
 
356         return __send__(match.target, match.attr_name, *args, &block)
 
357       end
 
358       super
 
359     end
